# README: MATERIAIS DE APRENDIZADO DE PYTHON
# Guia completo para iniciantes em Python

## Arquivos Criados

1. **01_conceitos_basicos.py** - Fundamentos do Python
   - Print e saída de dados
   - Variáveis e tipos de dados
   - Operações matemáticas
   - Entrada de dados (input)
   - Conversão entre tipos
   - Operadores de comparação
   - Operadores lógicos

2. **02_estruturas_controle.py** - Estruturas de controle
   - Condicionais (if/elif/else)
   - Laços de repetição (for e while)
   - Compreensão de listas
   - Menu interativo
   - Estruturas aninhadas

3. **03_funcoes_escopo.py** - Funções e escopo
   - Criação de funções
   - Parâmetros e retorno
   - Valores default
   - Argumentos nomeados
   - Número variável de argumentos
   - Funções lambda
   - Escopo de variáveis
   - Funções aninhadas
   - Recursão

4. **04_listas_estruturas.py** - Estruturas de dados
   - Listas e métodos
   - Tuplas
   - Dicionários
   - Conjuntos (sets)
   - List comprehension
   - Estruturas aninhadas
   - Métodos de strings

5. **05_exercicios_praticos.py** - Exercícios práticos
   - Calculadora simples
   - Verificador de números primos
   - Sistema de notas
   - Jogo de adivinhação
   - Gerenciador de tarefas
   - Conversor de temperaturas
   - Análise de texto
   - Validador de CPF
   - Jogo da forca
   - Sistema bancário

## Como Usar

1. **Execute os arquivos em ordem:**
   ```bash
   python 01_conceitos_basicos.py
   python 02_estruturas_controle.py
   python 03_funcoes_escopo.py
   python 04_listas_estruturas.py
   python 05_exercicios_praticos.py
   ```

2. **Estude cada conceito antes de prosseguir**

3. **Pratique com os exercícios no último arquivo**

## Dicas de Aprendizagem

1. **Leia o código antes de executar**
2. **Modifique os valores para ver o que acontece**
3. **Tente criar seus próprios exemplos**
4. **Use os exercícios para testar seu conhecimento**
5. **Não tenha medo de errar - é assim que se aprende!**
